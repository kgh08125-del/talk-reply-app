# 말투 답장 추천 웹앱

카카오톡 대화 내보내기 txt를 붙여넣으면 내 말투를 분석하고, 새 메시지에 맞는 답장 3개를 추천하는 로컬 웹앱입니다.

## 폴더 구조

```text
.
├─ index.html
├─ server.js
├─ manifest.json
├─ sw.js
├─ icons/
│  ├─ icon-192.png
│  └─ icon-512.png
├─ .env.example
└─ README.md
```

## 1. 준비물

Node.js 18 이상이 필요합니다.

터미널에서 아래 명령어를 입력해 버전이 나오면 준비된 상태입니다.

```bash
node -v
```

## 2. Gemini API 키 만들기

1. 브라우저에서 `https://aistudio.google.com/app/apikey` 로 이동합니다.
2. Google 계정으로 로그인합니다.
3. `Create API key` 또는 `API 키 만들기` 버튼을 누릅니다.
4. 만들어진 API 키를 복사합니다.

Gemini API 키는 보통 `AIza`로 시작합니다.

## 3. API 키 넣기

1. 이 폴더의 `.env` 파일을 엽니다.
2. 아래 줄에서 `AIza-your-api-key-here` 부분만 본인의 Gemini API 키로 바꿉니다.

예시:

```env
GEMINI_API_KEY=AIza실제_Gemini_API_키
GEMINI_MODEL=gemini-2.5-flash
PORT=3000
```

API 키는 서버에서만 읽습니다. `index.html`에는 API 키가 들어가지 않습니다.

중요: `.env`를 수정한 뒤에는 서버를 껐다가 다시 켜야 합니다.

## 4. 실행 방법

이 프로젝트는 외부 npm 패키지 없이 실행됩니다.

```bash
node server.js
```

브라우저에서 아래 주소를 엽니다.

```text
http://localhost:3000
```

## 5. 사용 방법

1. 카카오톡에서 대화 내보내기로 받은 txt 내용을 복사합니다.
2. 웹앱의 첫 번째 입력창에 붙여넣고 `대화 파싱하기`를 누릅니다.
3. 참여자 목록에서 내 이름을 선택합니다.
4. 최근 대화 미리보기를 확인하고 `말투 분석하기`를 누릅니다.
5. 원하는 분위기를 선택하거나 직접 뉘앙스를 입력합니다.
6. 상대방의 새 메시지를 입력하고 `답장 추천받기`를 누릅니다.
7. 마음에 드는 답장은 `복사`하거나 `이걸로 답장`을 눌러 이어지는 대화 흐름에 넣습니다.
8. 상대가 이어서 보낸 말을 다시 입력하면 3~4회 정도 자연스럽게 핑퐁 대화를 이어갈 수 있습니다.

## 지원하는 카카오톡 형식

PC 내보내기 예시:

```text
2026년 7월 5일 오후 9:12, 이름 : 메시지
```

모바일 내보내기 예시:

```text
[이름] [오후 9:12] 메시지
```

여러 줄 메시지는 앞 메시지에 이어붙여 처리합니다. 입장/퇴장 같은 시스템 메시지는 분석 대상에서 제외합니다.

## 오류 안내

- `API 키 설정`: `.env`에 `GEMINI_API_KEY`가 없거나 잘못된 경우입니다.
- `외부 API 연결`: 인터넷 연결, 방화벽, Gemini API 접속 상태를 확인해주세요.
- `API 오류`: 사용량 한도, 결제, 모델명, 권한 문제일 수 있습니다.
- `JSON 파싱 오류`: AI가 예상한 JSON 형식이 아닌 응답을 준 경우입니다. 다시 시도해보세요.

## 폰에서 앱처럼 설치하기

같은 네트워크에서 PC의 로컬 주소로 접속할 수 있게 설정하면, 모바일 브라우저에서 홈 화면에 추가할 수 있습니다. 이 앱에는 `manifest.json`, 서비스 워커, 192x192 및 512x512 아이콘이 포함되어 있습니다.
